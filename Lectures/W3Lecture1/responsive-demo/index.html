Responsive Layouts

Not Responsive

This page does not have any responsive styling. Most element widths are relative (using % or vw rather than pixels), which means that all content will still be visible on small screens. However, scaling will make much of the content difficult to read and interact with.

Columns are created using the inline-block approach.

Try each of the following:

  • Make your browser window narrow (less than 600px).
  • Make the text Very Large in Chrome preferences.
  • View the page on small devices in the device simulator.
