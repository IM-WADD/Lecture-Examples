Responsive Layouts

Flexbox

This page uses flexbox to create columns instead of the inline-block approach. Notice that different classes are used in this file than in the non-responsive and media queries pages. Flexbox does not remove the need for media queries, which are still in use to adjust the layout when the device width is less than 600px.

Try each of the following:

  • Make your browser window narrow (less than 600px).
  • Make the text Very Large in Chrome preferences.
  • View the page on small devices in the device simulator.
