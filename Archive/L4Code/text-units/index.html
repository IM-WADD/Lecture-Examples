This Section Uses Pixels For Everything

The default font size for paragraphs is usually 16px. The CSS for this section sets the paragraph font size to 16px and the heading font size to 64px, 4 times the paragraph size.

Browsers allow users to change the default font size of webpages. This is important for accessibility purposes. However, this functionality only works if CSS controlling font sizes is following the best practice of using relative units.

In Chrome, go to Preferences > Appearance and change the font size to "very large" then "very small". Notice how the text on this page does or doesn't change in line with the preference change. When the preference is "medium" (browser default), the headings and paragraphs in each section should be the same size.

This Section Uses Relative Units

The CSS for the previous section uses fixed pixel units for the font sizes. This section uses relative units. The paragraph setting is 1em, which equates to the browser setting for paragraphs times 1. The heading setting is 4em, which equates to the browser setting for paragraphs times 4.
