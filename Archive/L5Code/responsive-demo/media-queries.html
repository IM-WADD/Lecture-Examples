Responsive Layouts

Media Queries

This page adds the viewport meta tag to prevent scaling. Media queries are used to create a fluid layout. When the width of the screen is less than 600px, some font sizes and margins are adjusted, decorative elements are not displayed, and the menu layout is changed.

Columns are still created using the inline-block approach and all classes are the same as in the non-responsive page.

Try each of the following:

  • Make your browser window narrow (less than 600px).
  • Make the text Very Large in Chrome preferences.
  • View the page on small devices in the device simulator.
